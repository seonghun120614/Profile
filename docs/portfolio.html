NU Short Term Program

January 2024 — Feburary 2024

During a short-term program at the University of Northumbria, I conducted a project on transparent object localization, which lasted for two weeks. Utilizing PyTorch, I personally developed the model, incorporating the core idea of ResNet into the U-Net architecture and implementing it simply. While the evaluation wasn't outstanding, I gained insights into internal computational processes by addressing issues such as dimension mismatches during model development, which I hadn't been familiar with previously. As a result, I achieved the top position in the short-term project.

MyCanvas

September 2023 - December 2023

This project was conducted during the Data Structures course in my sophomore year, where we created a simple drawing application. The project involved implementing functionalities such as drawing lines, triangles, rectangles, circles, as well as features like color changing, grouping, shape color modification, undo & redo, save & load. In this project, I designed the rough relationships between the objects (ERD) and provided direction for my team members based on the proposed framework.

ICCV 2023 DFAD

September 2023 — October 2023

ELSA's Deep Fake Detection is a project aimed at distinguishing between fake and real images due to the vast data generation of generative AI. It was initiated to seize opportunities for presentations at conferences and personal growth. Leveraging Docker and Jupyter Notebook in a Linux server environment, approximately 1.5 million fake and real images were processed. However, the training process was significantly slow. This challenge was overcome by utilizing pytorch's amp.

Woowacourse Pre-course

November 2022 - December 2022

Woowacourse Pre-course is a preliminary test conducted to assess eligibility for participating in the Woowacourse regular curriculum. This test requires a deep understanding of Java and involves solving three problems structured in the Test-Driven Development (TDD) approach. Each problem involves creating games related to baseball, bridge, and lotto. I collaborated with professionals and other individuals experienced in Java to solve the problems and share our approaches. Throughout the process, I familiarized myself with package naming conventions for package separation, function name snippets for collaboration, and core features of Object-Oriented Programming (OOP).
